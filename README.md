## 📖 About

Welcome to the **Data Engineering Laboratory** repository.

This repository is designed for **Data Science students** to learn Data Engineering from scratch through theory, laboratory experiments, executable Python programs, Apache Airflow, Apache NiFi, PostgreSQL, Elasticsearch, and Kibana.

---

## 🎯 Objectives

After completing this laboratory, students will be able to

- Understand Data Engineering concepts
- Read and write different types of files
- Connect Python with PostgreSQL
- Build ETL pipelines
- Create Apache Airflow DAGs
- Develop Apache NiFi flows
- Build data engineering projects
