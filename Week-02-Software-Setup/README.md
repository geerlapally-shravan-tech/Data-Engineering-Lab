# Week 2 – Installation and Configuration

## Experiment Title

Installation and Configuration of Elasticsearch, Kibana, PostgreSQL and pgAdmin4

---

## Objectives

After completing this experiment, students will be able to:

- Understand the role of PostgreSQL in Data Engineering.
- Install PostgreSQL Database Server.
- Install pgAdmin4.
- Install Elasticsearch.
- Install Kibana.
- Verify the successful installation of all software.

---

## Software Covered

- PostgreSQL
- pgAdmin4
- Elasticsearch
- Kibana
