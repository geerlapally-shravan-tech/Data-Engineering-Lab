# Week 5 – Cleaning, Transforming and Enriching Data

## Experiment Title

Cleaning, Transforming and Enriching Data

---

## Objectives

After completing this experiment, students will be able to:

- Understand data quality issues.
- Handle missing values.
- Remove duplicate records.
- Transform raw data into a usable format.
- Enrich datasets with additional information.
- Perform data cleaning using Python and Pandas.
- Automate cleaning workflows using Apache Airflow.

---

## Topics Covered

- Missing Values
- Duplicate Records
- Data Cleaning
- Data Transformation
- Data Enrichment
- Pandas
- Apache Airflow
