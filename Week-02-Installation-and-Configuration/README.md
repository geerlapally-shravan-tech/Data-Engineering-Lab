# Week 02 – Installation and Configuration

## Experiment Title

**Installing and Configuring Elasticsearch, Kibana, PostgreSQL, and pgAdmin4**

---

## 📖 Introduction

Modern Data Engineering platforms require databases and visualization tools to efficiently store, search, analyze, and visualize data. PostgreSQL is a powerful open-source relational database management system, while pgAdmin4 is its graphical administration tool. Elasticsearch is a distributed search and analytics engine, and Kibana is a visualization platform used to create dashboards and analyze data stored in Elasticsearch.

In this experiment, students will install, configure, and verify PostgreSQL, pgAdmin4, Elasticsearch, and Kibana on their local systems. These tools form the foundation for data storage, search, analytics, and visualization in Data Engineering.

---

## 🎯 Aim

To install, configure, and verify Elasticsearch, Kibana, PostgreSQL, and pgAdmin4 for Data Engineering applications.

---

## 🎯 Objectives

After completing this experiment, students will be able to:

- Understand the purpose of PostgreSQL.
- Understand the role of pgAdmin4.
- Understand Elasticsearch and its architecture.
- Understand Kibana dashboards.
- Install PostgreSQL Database Server.
- Install pgAdmin4.
- Install Elasticsearch.
- Install Kibana.
- Verify successful installation of all software.

---

## 🎓 Learning Outcomes

Students will be able to:

- Install PostgreSQL successfully.
- Create databases using pgAdmin4.
- Install Elasticsearch.
- Start Elasticsearch services.
- Install Kibana.
- Connect Kibana with Elasticsearch.
- Verify the complete Elastic Stack installation.
