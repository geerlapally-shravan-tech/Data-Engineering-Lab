# Installation Guide

## Step 1

Install PostgreSQL

Verify

```bash
psql --version
```

---

## Step 2

Install pgAdmin4

Open pgAdmin

Connect to PostgreSQL

---

## Step 3

Install Elasticsearch

Extract ZIP

Run

```text
elasticsearch.bat
```

Verify

```
http://localhost:9200
```

---

## Step 4

Install Kibana

Extract ZIP

Run

```text
kibana.bat
```

Open

```
http://localhost:5601
```

---

## Step 5

Verify all software

- PostgreSQL
- pgAdmin4
- Elasticsearch
- Kibana
