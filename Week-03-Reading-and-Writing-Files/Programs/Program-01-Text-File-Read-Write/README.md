# Program-01

# Reading and Writing Text Files using Python

---

## Aim

To perform reading and writing operations on text files using Python.

---

## Objective

The objective of this experiment is to understand how Python performs file handling operations such as creating, reading, writing, appending, and closing text files.

---

## Prerequisites

- Python 3.x
- VS Code
- Basic knowledge of Python

---

## Theory

A text file is the simplest way to store information permanently. Python provides built-in functions to create, read, write, and modify text files.

The `open()` function is used to open or create a file.

After processing the file, it should always be closed using the `close()` function.

---

## Algorithm

1. Start
2. Open a text file in write mode.
3. Write student information.
4. Close the file.
5. Open the file in read mode.
6. Read the contents.
7. Display the contents.
8. Close the file.
9. Stop.
