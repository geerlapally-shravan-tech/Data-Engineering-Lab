# Programs

# Week 03 – Reading and Writing Files

---

## Experiment No. 3

Reading and Writing Files

---

## Objective

The objective of this experiment is to understand Python file handling techniques for reading and writing data using different file formats and PostgreSQL databases.

---

## Programs Included

| Program No | Program Title | Description |
|-------------|---------------|-------------|
| Program-01 | Reading and Writing Text Files | Create, write, read, and append text files |
| Program-02 | Reading and Writing CSV Files | Read and write CSV files using the csv module |
| Program-03 | Reading and Writing JSON Files | Read and write JSON files using the json module |
| Program-04 | Reading and Writing PostgreSQL Database | Connect Python with PostgreSQL and perform CRUD operations |

---

## Software Required

- Python 3.x
- Visual Studio Code
- PostgreSQL
- pgAdmin4
- psycopg2
- CSV Module
- JSON Module

---

## Learning Outcomes

After completing these programs, students will be able to:

- Perform text file operations.
- Read and write CSV files.
- Read and write JSON files.
- Connect Python with PostgreSQL.
- Store and retrieve records from databases.
- Understand the role of file handling in Data Engineering.

---

## Program Sequence

Students should complete the programs in the following order:

1. Text File Handling
2. CSV File Handling
3. JSON File Handling
4. PostgreSQL Database Connectivity

Each program builds on the concepts learned in the previous one.

---

## Expected Outcome

After completing all four programs, students will have a strong understanding of Python file handling and database operations, forming the foundation for ETL processes and Data Engineering pipelines.
