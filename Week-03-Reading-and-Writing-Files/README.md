# Week 03 – Reading and Writing Files

## Experiment Title

**Reading and Writing Files**

---

## 📖 Introduction

Data Engineers frequently work with different file formats to store, exchange, and process data. Python provides built-in libraries to handle text files, CSV files, and JSON files, while database libraries such as `psycopg2` enable interaction with PostgreSQL databases.

In this experiment, students will learn how to create, read, update, and write data using different file formats and store structured data in a PostgreSQL database.

---

## 🎯 Aim

To perform file handling operations using Python by reading from and writing to text files, CSV files, JSON files, and PostgreSQL databases.

---

## 🎯 Objectives

After completing this experiment, students will be able to:

- Understand Python file handling.
- Read and write text files.
- Read and write CSV files.
- Read and write JSON files.
- Connect Python with PostgreSQL.
- Store and retrieve records from a database.

---

## 🎓 Learning Outcomes

Students will be able to:

- Perform file operations using Python.
- Work with structured and semi-structured data.
- Connect Python applications to PostgreSQL.
- Build the foundation for ETL pipelines.
